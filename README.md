# SimpleNN
Simple Neural network implemented in C++, trained using back propagation or genetic algorithm.

Examples (back prop): XOR, sine, MNIST (csv from http://pjreddie.com/projects/mnist-in-csv/)
Examples (ga): XOR, snake
